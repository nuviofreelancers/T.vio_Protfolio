The problem

Recruiters lose time before they can make a decision.

Manual CV screening creates bottlenecks, inconsistent review quality, and missed information across messy resume formats.

The hiring friction

Recruiters waste hours screening CVs. Different file types and formats make it harder to extract the same information from every candidate.

  • Recruiters spend hours reviewing CVs manually
  • Inconsistent formats can hide important information
  • Manual review creates bias and bottlenecks
The solution

An AI screening platform that structures the hiring workflow.

TalentScreen parses resumes, extracts candidate data, scores applicants, and helps HR teams filter and compare results faster.

Core product direction

The platform uses AI to parse and score resumes, extract structured candidate data, and support fast filtering and side-by-side comparison.

  • Resume parsing for PDF, DOCX, Google Drive, and Docs links
  • AI screening with configurable weighted scoring
  • Smart filtering and candidate comparison

Technical deep dive

Architecture, stack, features, and challenges.

This section explains the technical thinking behind the build and the decisions that shaped the product.

Frontend architecture

React + Vite powers the UI, with Redux handling applicants, jobs, and results. The API client connects the interface to the backend, while AuthContext manages JWT auth in localStorage.

Backend architecture

Express + TypeScript provides protected routes, JWT middleware, and controllers for applicants, jobs, screening, and authentication.

AI and parsing pipeline

The AI service extracts CV data and screens applicants using Gemini 2.5 Flash, with file parsing support through pdf-parse, mammoth, and tesseract.

Key challenges

The team handled inconsistent resume formats, noisy Drive link data, prompt tuning, validation, and bias detection so HR users could trust the results.

Impact

Faster, fairer hiring decisions.

The product replaces guesswork with structured candidate data and surfaces bias flags to support more thoughtful review.

Project outcomes

  • Faster filtering and candidate comparison
  • Structured data replaces manual guesswork
  • Bias flags surfaced to HR users
